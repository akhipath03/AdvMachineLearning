Nutrition Insight Mobile App

Developed an intuitive mobile application with React Native, enabling users to capture ingredient labels via camera or their phone's gallery.

  • Implemented seamless user authentication through AWS Cognito, providing personalized experiences based on attributes like age, allergies, and health concerns.
  • Utilized AWS Lambda for efficient image uploads and processing, integrating Lambda, S3, and Textract for streamlined image analysis and text extraction.
  • Utilized OpenAI API to dynamically generate personalized and detailed insights about extracted ingredients, ensuring users are informed about potential harmful substances in products.

Technologies used: Node.js, React-Native, AWS, OpenAI API

GitHub Repository: Nutrition Insight Mobile App

New Jersey Residential Price Projection System

Conducted data cleaning and analysis on over 700k housing instances using Jupyter Notebooks.

  • Preprocessed data with one-hot encoding for categorical features (e.g., zip codes), data imputation for handling missing values, and MinMaxScaler normalization. Split data for training and validation.
  • Developed Sequential neural network with callbacks specialized Dense and Dropout layers, alongside hyperparameter-tuned Random Forest Regressor.
  • Achieved over 75% accuracy with both models in forecasting property prices.

Technologies used: Python, Jupyter Notebooks, Tensorflow, NumPy, Pandas, Scikit-learn

GitHub Repository: New Jersey Residential Price Projection System

Global Food Systems and Gas Emissions

My final project in my data visualization class aimed to "show the whole story" of global food systems and their contribution to gas emissions.

I created an HTML website featuring multiple interactive visualizations using Tableau. The project delved into the relationship between global food systems and gas emissions, presenting the impact at both country and food-specific levels.

The visualizations included:

  • Choropleth map showing greenhouse gas emissions by country.
  • Stacked bar chart highlighting emissions from all stages of food production per food.
  • Bar chart categorizing emissions by food groups.

After formatting and adding an introduction and conclusion for context and main takeaways, I enhanced the user experience with a CSS stylesheet.

Website: Global Food Systems and Gas Emissions

Gradient Boosting Lowess vs eXtreme Gradient Boosting Library

Created a class implementing the Gradient Boosting concept based on the locally weighted regression method (Lowess class), allowing a user-prescribed number of boosting steps. The class includes mainstream useful options such as "fit," "is_fitted," and "predict" methods.

Demonstrated applications with real data for regression, 10-fold cross-validations, and compared the effect of different scalers including the "StandardScaler," "MinMaxScaler," and the "QuantileScaler." In the case of the "Concrete" dataset, determined a choice of hyperparameters that yielded lower Mean Squared Errors (MSEs) for the implemented method compared to the eXtreme Gradient Boosting library.

Technologies used: Python, NumPy, Pandas, Scikit-learn, xGBoost

GitHub Repository: Gradient Boosting Lowess vs eXtreme Gradient Boosting Library

KNN Regressor using USearch Library

Developed a KNN Regressor using the USearch Library in Python. The KNN Regressor is implemented to predict the labels of data points based on the labels of their nearest neighbors. The implementation allows for a user-defined number of nearest neighbors (k).

Demonstrated the performance of the KNN Regressor using the iris dataset from the UCI Machine Learning Repository. Preprocessed the data, split it into training and testing sets, and evaluated the model's accuracy across varying values of k.

Technologies used: Python, NumPy, Pandas, Scikit-learn, USearch
